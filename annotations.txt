1. Install Pulumi
>> choco install pulumi
>> pulumi version

2. Install the AWS CLI and PowerShell's helper
>> choco install awscli -y (CMD admin)
>> Import-Module "$env:ChocolateyInstall\helpers\chocolateyProfile.psm1" (PowerShell)
>> refreshenv
>> aws --version

3. Install eksctl
>> choco install eksctl -y
>> eksctl version

4. If any secrets/confidential files where accidentally pushed to GitHub, rever the changes with:
>> pip install git-filter-repo (CMD)
>> python -m git_filter_repo --path secrets.txt --invert-paths --force (PowerShell)
